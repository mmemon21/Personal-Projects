msh1.c: A simple bash shell that responds to commands 'exit', 'help' & 'today'.
msh2.c: This shell has a few added features. It can carry out bash built in commands such as ls.
msh3.c: This shell contains features from both msh1 and msh2 as well as responds accordingly to cd and also can take a command input from 
        an external file.
